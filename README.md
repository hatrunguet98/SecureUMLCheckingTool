# Secure-UML-Checking-Tool
Tool check RBAC for application

# Version
<ul>
<li>Java: 8</li>
<li>Maven: 3.6.1</li>
<li> Swing GUIs</li>
<li> Netbeans IDE 8.2</li>
</ul>

# Install
mvn clean compile assembly:single

# Run
<ul>
<li>cd target</li>
<li>java -jar SecureUMLCheckingTool-1.0-SNAPSHOT-jar-with-dependencies.java</li>
</ul>
